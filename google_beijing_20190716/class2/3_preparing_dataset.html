3. 数据预处理

3.1 将标签转化为one-hot编码

在原始数据中，鸢尾花的类别是字符型，在进行机器学习时，需要转化为数值型。在本项目中，涉及到三个种类的鸢尾花，我们对鸢尾花的标签进行 one-hot 编码。

采用 one-hot 编码的原因，可以参考 什么是one hot编码？为什么要使用one hot编码？

选中 2.2 将标签转化为one-hot编码 单元格，插入新的代码单元格，输入以下代码：

# 对species进行one-hot编码
one_hot_species = pd.get_dummies(iris['species'], prefix='species')
# 随机输出5行，查看效果
one_hot_species.sample(5)

一个可能的运行结果是：

get_dummies是pandas实现one-hot编码的方式，prefix参数用来指定增加列的前缀。

继续增加新的代码单元格，输入以下代码：

#  与原来的数据进行拼接
one_hot_data = pd.concat([iris,one_hot_species], axis=1)
# 随机输出5行，查看效果
one_hot_data.sample(5)

一个可能的运行结果是：

concat函数用来拼接pandas的对象（dataframe等），参数axis用来指定拼接的方向，axis=1代表横行拼接

继续增加新的代码单元格，输入以下代码：

#  删除原来的species列
one_hot_data = one_hot_data.drop('species', axis=1)
# 随机输出5行，查看效果
one_hot_data.sample(5)

drop用来删除特定的行或者列。参数axis用来指定删除的方向，axis=1代表删除行

一个可能的运行结果是：

到此为止，我们就得到了一个将标签转换为one-hot编码的数据集。

删除不必要的输出，将上面的三段代码整理如下：

#  对species进行one-hot编码
one_hot_species = pd.get_dummies(iris['species'], prefix='species')
#  原来的数据进行拼接
one_hot_data = pd.concat([iris,one_hot_species], axis=1)
#  删除原来的species列
one_hot_data = one_hot_data.drop('species', axis=1)
#  随机输出5行，查看效果
one_hot_data.sample(5)

3.2 分割特征和标签

选中 2.3 分割特征和标签 单元格，插入新的代码单元格，输入以下代码：

# 提取特征，前4列为特征
features = one_hot_data.iloc[:,0:4]
print("\n=====显示前5行特征=====")
print(features.head())

iloc用于通过索引来获取dataframe中的行和列，在上面的代码中，iloc后面通过切片运算获取了所有的行的前四列（第一个：代表所有行，逗号后的0:4代表前4列），

运行结果如下：

# 提取标签，后3列为标签
labels = one_hot_data.iloc[:,4:]
print("\n=====显示前5行标签=====")
print(labels.head())

运行结果如下：

将特征和标签从dataframe数据类型转换为numpy array数据类型，便于在模型中运算

features = np.array(features)
labels = np.array(labels)

查看一下，特征和标签的形状
