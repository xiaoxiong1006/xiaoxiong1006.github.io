4.数据的探索

4. 探索数据

在本单元中，将从 数据类型 ，数据内容 两个方面进行数据的探索。

4.1 数据内容的探索

从测试集中，任取一张数字的图片，输出在该图片的 类型 、维度 和 大小 等相关类型信息。

选中 3.1 数据类型的探索 的单元格，插入新的代码单元格，输入以下语句：

idx = 1046
img = mnist.test.images[idx]
print("数据类型：", type(img))
print("数组元素的数据类型",img.dtype)
print("数组元素的总数",img.size)
print("数组元素的形状",img.shape)
print("数组元素的维度",img.ndim)

idx 的取值为 [1,10000] 的任意整数

输出结果为：

可以看到，从 MNIST 中分离出的 测试集 test 的 image 部分是 numpy.ndarray 类型，是 784*1 的数组，数组元素为 float32

4.2 数据内容的探索

1.图像（特征）部分的探索

遍历 image 数组，按照 28行28列 的形式输出数组内容:

选中 3.2 数据内容的探索 的单元格，插入新的代码单元格，输入以下语句：

# 图像（特征）部分的探索
count = 0
for i in img:
    print("%3d" %(int(i*255)),end=" ")
    count += 1
    if(count % 28 == 0):
        print(end="\n")

2.标签部分的探索

接下来探索标签数据，继续插入新的代码单元格，输入以下语句：

# 标签部分的探索
label = mnist.test.labels[idx]
print(label)

运行结果如下：

可以看出，在标签部分采用了 one-hot 编码，在第 6 个位置的值为 1 ，代表数字 5 。（第一个分类是 0 ）

3.可视化

将图片和标签通过 matplotlib.pyplot 进行可视化。

继续插入新的代码单元格，输入以下语句：

# 可视化
plt.title(label)
img = img.reshape(28,28)
plt.imshow(img,cmap=plt.get_cmap('gray_r'))

4.3 结论

可以看出，在数据集的 image 部分中存放着数字的图片，在 label 部分存放着对应图片的 one-hot 编码标签。
