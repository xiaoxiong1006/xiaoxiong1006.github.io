9.导入模型和使用模型

9. 使用模型

9.1 模型的导入

增加一个导包：

  • load_model

在程序开头的导包单元格，增加：

from keras.models import load_model

完整的导包代码如下：

# 导包
import tensorflow as tf
from tensorflow.examples.tutorials.mnist import input_data
import numpy as np
import matplotlib.pyplot as plt
from keras.models import load_model

重新运行导包单元格。

选中 8.1 导入保存好的模型 单元格，插入新的代码单元格，输入以下代码：

model_saved = load_model("model/model_mnist_linear.h5")

运行，本单元格没有输出，不报错即可。

9.2 模型的使用

找测试集中任取一张图片，并可视化。

选中 8.2 在测试集上测试 单元格，插入新的代码单元格，输入：

# 在测试集上任取一张图片
idx = np.random.randint(10000) #产生一个1～10000的随机数作为下标
img = mnist.test.images[idx]
img = img.reshape(28,28)
plt.imshow(img,cmap=plt.get_cmap('gray_r'))
