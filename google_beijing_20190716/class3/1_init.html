2.准备工作

2. 准备工作

进入workspace，在该目录下新建名为 datasets 的目录用于存放数据集。

切换到 Terminal 界面，进入 workspace 目录，复制数据集文件到 workspace/datasets/ 目录下:

cp -r ../Tools/datasets/* datasets/

复制模板文件到本目录下:

cp -r ../Tools/01_mnist_linear_softmax_templete.ipynb 01_mnist_linear_softmax.ipynb

双击打开文件 01_mnist_linear_softmax.ipynb，首先进行依赖包的导入。需要导入以下依赖包：

  • numpy
  • matplotlib.pyplot
  • tensorflow
  • tensorflow.examples.tutorials.mnist 中 input_data.py 文件提供了下载MNIST数据集的python代码
## 导包
import numpy as np
import matplotlib.pyplot as plt
import tensorflow as tf
from tensorflow.examples.tutorials.mnist import input_data
