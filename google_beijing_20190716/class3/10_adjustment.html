10.改进模型

10. 改进模型

我们前面使用的 线性模型 + softmax 在MNIST上只有91%正确率，在实际场景中，正确率不到70%。因此，我们用一个稍微复杂的模型：卷积神经网络 来改善效果。这会达到大概99.2%的准确率。

在原来代码的基础上，

10.2 对数据预处理的调整

# 输入图片的维度
img_rows, img_cols = 28, 28

## 训练集
X = mnist.train.images.reshape(-1,img_rows,img_cols)
Y = mnist.train.labels

## 验证集
X_validation = mnist.validation.images.reshape(-1,img_rows,img_cols)
Y_validation = mnist.validation.labels

## 测试集
X_test =  mnist.test.images.reshape(-1,img_rows,img_cols)
Y_test =  mnist.test.labels

if K.image_data_format() == 'channels_first':
    X = X.reshape(X.shape[0], 1, img_rows, img_cols)
    X_validation = X_validation.reshape(X_validation.shape[0], 1, img_rows, img_cols)
    X_test = X_test.reshape(X_test.shape[0], 1, img_rows, img_cols)
    input_shape = (1, img_rows, img_cols)
else:
    X = X.reshape(X.shape[0], img_rows, img_cols, 1)
    X_validation = X_validation.reshape(X_validation.shape[0], img_rows, img_cols, 1)
    X_test = X_test.reshape(X_test.shape[0], img_rows, img_cols, 1)
    input_shape = (img_rows, img_cols, 1)

print("训练集特征的形状：" + str(X.shape))

10.3 对模型的调整

将原来 搭建模型 的代码替换为：

# 搭建模型
model = Sequential()
model.add(Conv2D(32, kernel_size=(3, 3),activation='relu',input_shape=input_shape))
model.add(Conv2D(64, (3, 3), activation='relu'))
model.add(MaxPooling2D(pool_size=(2, 2)))
model.add(Flatten())
model.add(Dense(128, activation='relu'))
model.add(Dense(10, activation='softmax'))
model.summary()  # 查看模型结构
