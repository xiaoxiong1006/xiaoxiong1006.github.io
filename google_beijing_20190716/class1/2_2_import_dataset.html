2.2 导入数据

2.2.1 复制模版文件

在这个 Codelab 中，提供完成项目的ipynb文件的模板。 切换到 Terminal 界面，在 Codelab 目录中，复制模板文件到本目录下。

cp -r ../Tools/01_iris_bi_classification_templete.ipynb 01_iris_bi_classification.ipynb

2.2.2 导入依赖包

双击打开文件 01_iris_bi_classification.ipynb ，首先进行 依赖包 的导入。

双击 导包 的单元格，导入Pandas科学计算库。关于Pandas的详细情况，可以参考Pandas的官网 或者 Pandas 中文文档

## 导包
import pandas as pd

DataFrame 是Pandas的基本数据结构之一。 DataFrame 是二维数据结构，即数据以行和列的表格方式排列。 Pandas DataFrame由三个主要组件组成，即 数据(data) , 行(rows) 和 列(columns) 。

2.2.3 导入数据集

选中 2.导入数据 的单元格，插入新的代码单元格，输入以下代码：

datasets_path = 'data/iris.data'
iris = pd.read_csv(datasets_path,header=None)
print(iris.head())

read_csv 是pandas用来读取和操作csv文件一个重要的函数。通常只需要给定csv文件的地址即可。 header 参数用来指定行数作为列名，None代表不使用文件中的任何一列作为列名。

head 函数用来返回DataFrame对象的前5行，通常用来预览读入的数据。

运行当前单元格，可以看到数据可以被正确读入，但是 列的标题 使用了 自动生成的数字 。

接下来，我们将其替换为 更有意义 的列名后，再输出。

在当前单元格的第三行，添加代码

iris.columns =["sepel_length",'sepel_width',"petal_length",'petal_width','species']

完整的代码如下：

datasets_path = 'data/iris.data'
iris = pd.read_csv(datasets_path,header=None)
iris.columns =["sepel_length",'sepel_width',"petal_length",'petal_width','species']
print(iris.head())
